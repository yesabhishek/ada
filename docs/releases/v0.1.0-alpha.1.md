# Ada v0.1.0-alpha.1

Public alpha release date: 2026-04-17

## What shipped

- local Go CLI for semantic sidecar workflows
- local `.ada/` SQLite storage for snapshots, symbols, sync runs, tasks, and proposals
- semantic diff for Go and TypeScript/JavaScript
- bounded semantic merge and safe materialization paths for supported languages
- local read-only UI for snapshot history, semantic drift, and sync health
- built-in `ada eval` scenarios for comparing Git and Ada merge behavior
- `ada version`, `ada doctor`, and JSON output for `status`, `diff`, and `eval`
- GitHub Actions CI, Goreleaser config, install script, and Homebrew tap publishing config

## Supported platforms

- macOS `amd64`
- macOS `arm64`
- Linux `amd64`
- Linux `arm64`

## What to try first

```bash
ada start .
ada sync
ada diff --semantic
ada ui --open
ada eval
```

## Known limitations

- Ada is a Git sidecar today, not a Git replacement
- `ada sync` requires a clean Git working tree
- only Go and TypeScript/JavaScript are supported in the public alpha
- the local CLI and local UI are the supported product surface
- remote/control-plane code is experimental and not part of the supported workflow
- Windows support, package-manager distribution, and hosted multi-user workflows are deferred
