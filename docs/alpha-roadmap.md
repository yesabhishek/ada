# Alpha Roadmap

## v0.1.x Alpha Goals

- make local onboarding reliable and fast
- stabilize CLI output and error behavior
- improve semantic diff and merge confidence for supported languages
- collect feedback on the local UI and eval framework

## Not In The First Alpha

- Windows support
- hosted multi-user control plane
- full Git replacement positioning
- languages beyond Go and TS/JS

## Next Hardening Priorities

- more real-world eval scenarios
- better JSON output coverage
- better diagnostic output from `ada doctor`
- expansion of supported semantic operations within a single symbol
